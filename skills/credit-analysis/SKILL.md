# 信用风险分析

## fiagent 可用工具

| 检测维度 | 工具 | 指标 |
|---------|------|------|
| 财务造假 | `get_financial_statements(statement="indicators")` | 12 个红旗（详见 financial-statement skill） |
| 质押爆仓 | 无专用工具，用 `get_stock_news` 监控质押公告 | 大股东质押比例 >80% = 高风险 |
| 商誉减值 | `get_financial_statements(statement="balance")` | 商誉/净资产 >30% = 预警 |
| 审计意见 | `get_financial_statements` + 公告搜索 | 非标准无保留意见 = 回避 |
| 退市风险 | `screen_fundamental(exclude_st=False)` 查看 ST 列表 | ST=已触发，*ST=退市风险警示 |

## 关键阈值

| 风险指标 | 安全 | 关注 | 危险 |
|---------|------|------|------|
| 资产负债率 | <50% | 50-70% | >70% |
| 有息负债/总资产 | <20% | 20-40% | >40% |
| 速动比率 | >1.0 | 0.5-1.0 | <0.5 |
| CFO/净利润 | >1.0 | 0.5-1.0 | <0.5 |
| 大股东质押比例 | <50% | 50-80% | >80% |
| 商誉/净资产 | <10% | 10-30% | >30% |

## 可转债信用筛选

```
screen_cb(min_rating="AA", max_double_low=130)
→ 筛掉 AA 级以下转债（信用风险过滤）
```

## A 股特色信用事件

1. **股权质押爆仓**：大股东质押比例 >80% + 股价跌至平仓线 → 强制平仓 → 加速下跌
2. **商誉暴雷**：并购三年业绩承诺到期后商誉减值 → 一次性巨亏
3. **退市新规**：连续 20 日股价 <1 元（面值退市）、营收 <1 亿且净利润为负
4. **债券违约**：民营房企/城投平台的信用事件传染效应
