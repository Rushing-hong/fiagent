## 基差分析

```
基差 = 现货价格 - 期货价格
正基差(现货升水) = 现货紧缺 → 做多近月
负基差(期货升水) = 库存充足 → 关注展期收益
```

## 产业链利润

```
螺纹钢利润 = 螺纹钢 - 1.6×铁矿 - 0.5×焦炭 - 800
利润<0: 钢厂减产 → 利多螺纹
利润>500: 暴利增产 → 利空螺纹
```

## 库存周期

```
主动去库 → 被动去库(最佳做多) → 主动补库 → 被动补库(最佳做空)
```
