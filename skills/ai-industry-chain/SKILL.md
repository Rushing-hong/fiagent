# AI 算力产业链 A 股映射

> 2026 年上半年 AI 产业链是绝对主线。TMT 成交占全市场 40%+，通信板块涨 55.9%。

## 用 `get_market_breadth` 监控拥挤度

```
get_market_breadth(mode="tmt_only")
→ TMT 成交占比: 35% = 极度拥挤 / 30% = 高度拥挤 / 20% = 温和
```

## 监控方法

```
1. get_market_breadth(mode="full")
   → 看 TMT 占比和拥挤度

2. screen_market(sort_by="change_pct", top_n=30)
   → AI 相关板块涨跌幅排行

3. get_etf_flow(sort_by="net_inflow")
   → 通信 ETF/半导体设备 ETF 资金流向

4. get_fund_flow(codes=["300308.SZ","002463.SZ","688256.SH"], period="daily")
   → 核心标的资金流向
```
