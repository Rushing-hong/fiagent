## 财经新闻（Stock / Global News）

工具：`get_stock_news`（`tools.stock_research.StockNewsTool`）
描述：东财免费 `search-api` 新闻列表。按源 IP 限流，走共享 eastmoney 节流客户端。
- **scope=stock**（默认）：按个股代码关键词检索（取 `.` 前裸代码作 keyword）。
- **scope=global**：宽口径财经头条，默认查询词 `财经`。
市场：A 股（`.SH` / `.SZ` / `.BJ`）。
限速：东财经 `eastmoney` bucket。

<br>

**端点**

| 上游 | 端点 URL |
| ---- | -------- |
| 东财新闻 | `https://search-api-web.eastmoney.com/search/jsonp` |

<br>

**输入参数（工具入参）**

名称 | 类型 | 必选 | 描述
---- | ---- | ---- | ----
scope | str | N | `"stock"`（默认，需 `code`）或 `"global"`（宏观财经，无需 `code`）。
code | str | scope=stock 必选 | A 股 symbol，如 `600519.SH`。工具用裸代码作搜索词。
limit | int | N | 返回头条数上限（1–50，默认 20）。

<br>

**端点查询参数（东财）**

`cb=`、`_=0`、`param=<JSON 字符串>`。其中 `param` 体：`{"uid":"","keyword":<查询词>,"type":["cmsArticleWebOld"],"client":"web","clientType":"web","param":{"cmsArticleWebOld":{"searchScope":"default","sort":"default","pageIndex":1,"pageSize":<limit>}}}`。响应可能为 JSONP（`callback(...)`），工具自动剥壳。

<br>

**返回字段**

东财文章（`result.cmsArticleWebOld[]`，源字段 → 输出键）：

源字段 | 输出键 | 描述
---- | ---- | ----
title | title | 标题
url | url | 链接
mediaName | source | 媒体名
date | published | 发布时间
content | snippet | 正文（裁剪至 280 字符）

信封：`{"ok": true, "market": "cn", "source": "eastmoney", "data": {"query", "scope", "count", "articles":[...]}}`。

<br>

**调用范例**

```python
from tools.stock_research import StockNewsTool

print(StockNewsTool().execute({"code": "600519.SH", "scope": "stock", "limit": 10}, None))
print(StockNewsTool().execute({"scope": "global", "limit": 15}, None))
```
