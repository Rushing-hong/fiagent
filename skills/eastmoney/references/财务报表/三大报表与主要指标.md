## 三大报表与主要指标（Financial Statements）

工具：`get_financial_statements`（`tools.stock_research.FinancialStatementsTool`）
描述：取单只 **A 股** 的资产负债表 / 利润表 / 现金流量表 / 主要指标（东财 F10 datacenter）。逐期扁平行返回，并附带 `normalized` 标准字段（`revenue` / `net_income` / `cfo` 等），杜邦/红旗/DCF 应读 normalized，勿猜东财原始字段名。
市场：A 股（`.SH` / `.SZ` / `.BJ`）。
限速：东财经 `eastmoney` bucket。

<br>

**端点**

`https://datacenter.eastmoney.com/securities/api/data/v1/get`

**东财 reportName（statement → reportName）**

statement | reportName
---- | ----
balance | `RPT_F10_FINANCE_GBALANCE`
income | `RPT_F10_FINANCE_GINCOME`
cashflow | `RPT_F10_FINANCE_GCASHFLOW`
indicators | `RPT_F10_FINANCE_MAINFINADATA`

<br>

**输入参数（工具入参）**

名称 | 类型 | 必选 | 描述
---- | ---- | ---- | ----
code | str | Y | A 股 symbol，如 `600519.SH` / `000001.SZ`。
statement | str | N | `balance` / `income` / `cashflow` / `indicators`（默认 `indicators`）。
period | str | N | `annual`（年报）或 `quarter`（季报），默认 `annual`。

<br>

**端点查询参数**

`reportName=<上表>`、`columns=ALL`、`filter=(SECUCODE="<code>")`、`sortColumns=REPORT_DATE`、`sortTypes=-1`、`pageNumber=1`、`pageSize=40`、`source=F10`、`client=PC`。

<br>

**返回字段**

报表逐期行为扁平 `字段名 → 值`（东财原始字段）+ `normalized` 标准字段。防爆上限：每只股票最多 40 期。

信封：`{"ok": true, "market": "a_share", "source": "eastmoney", "statement": <statement>, "period": <period>, "data": {"code", "periods": [...]}}`。

<br>

**调用范例**

```python
from tools.stock_research import FinancialStatementsTool

print(FinancialStatementsTool().execute(
    {"code": "600519.SH", "statement": "balance", "period": "annual"}, None
))
print(FinancialStatementsTool().execute(
    {"code": "600519.SH", "statement": "indicators"}, None
))
```
