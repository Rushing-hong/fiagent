## 全市场选股（Market Screener）

工具：`screen_market`（`tools.stock_market.ScreenMarketTool`）
描述：A 股全市场行情列表。首选东财 push2 `clist`，失败依次降级 `push2delay`、`akshare.stock_zh_a_spot_em`。按所选字段（`fid`）服务端排序后取 top N。用于「今日涨跌幅/成交额前 N」等问法。
市场：仅 A 股（`market="a"`）。
限速：经共享 eastmoney per-host 节流；东财按源 IP 限流。降级到 akshare 时信封 `quality=degraded`。

<br>

**端点**

| 优先级 | URL |
| ---- | ---- |
| 1 | `https://push2.eastmoney.com/api/qt/clist/get` |
| 2 | `https://push2delay.eastmoney.com/api/qt/clist/get` |
| 3 | akshare `stock_zh_a_spot_em()`（无 HTTP clist） |

**市场全集选择器（fs）**

market | fs
---- | ----
a（沪深主板+创业板+科创+北交所） | `m:0+t:6,m:0+t:80,m:1+t:2,m:1+t:23,m:0+t:81+s:2048`

**排序字段（sort_by → fid）**：change_pct=`f3`、volume=`f5`、amount=`f6`、turnover=`f8`、pe=`f9`、market_cap=`f20`、pb=`f23`。

<br>

**输入参数（工具入参）**

名称 | 类型 | 必选 | 描述
---- | ---- | ---- | ----
market | str | Y | 仅 `"a"`（A 股）。
sort_by | str | N | `change_pct`（默认）/ `volume` / `amount` / `turnover` / `pe` / `pb` / `market_cap`。
top_n | int | N | 返回 top N（1–100，默认 30）。
ascending | bool | N | `false`（默认）降序=涨幅榜；`true` 升序=跌幅榜。映射东财 `po`：`0` 升 / `1` 降。

<br>

**端点查询参数（clist）**

`pn=1`、`pz=<top_n>`、`po=<0|1>`、`fid=<排序字段>`、`fs=<市场全集>`、`fields=f2,f3,f4,f5,f6,f8,f9,f12,f14,f20,f23`、`fltt=2`。

<br>

**返回字段**（field id → 输出键）

field id | 输出键 | 描述
---- | ---- | ----
f12 | code | 代码
f14 | name | 名称
f2 | price | 最新价
f3 | change_pct | 涨跌幅（%）
f5 | volume | 成交量（手）
f6 | amount | 成交额（元）
f8 | turnover_rate | 换手率（%）
f9 | pe | 市盈率-动态
f23 | pb | 市净率
f20 | market_cap | 总市值（元）

`-` 等哨兵值映射为 None。信封示例：`{"ok": true, "market": "a_share", "source": "eastmoney"|"akshare", "quality": "normal"|"degraded", "data": {"market","sort_by","ascending","count","stocks":[...]}}`。

<br>

**调用范例**

```python
from tools.stock_market import ScreenMarketTool

# 涨幅榜
print(ScreenMarketTool().execute(
    {"market": "a", "sort_by": "change_pct", "top_n": 20, "ascending": False}, None
))
# 跌幅榜
print(ScreenMarketTool().execute(
    {"market": "a", "sort_by": "change_pct", "top_n": 10, "ascending": True}, None
))
```
