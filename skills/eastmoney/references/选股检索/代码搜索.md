## 代码搜索（Symbol Search）

工具：`search_symbol`（`tools.stock_market.SearchSymbolTool`）
描述：把公司名或代码片段解析为 A 股候选 symbol（如 `600519.SH`）。单源：东财 suggest，经 IP 节流。
市场：面向 A 股检索（中文名 / 6 位代码）。suggest 若偶发返回其他市场号，工具会带对应后缀，但本技能约定只使用 `.SH` / `.SZ` / `.BJ`。
限速：东财经 `eastmoney` bucket。

<br>

**端点**

| 源 | 端点 |
| ---- | ---- |
| 东财 suggest | `https://searchapi.eastmoney.com/api/suggest/get`（`input=<query>`、`type=14`、`count=25`） |

**东财 market 号 → 后缀（工具映射）**：`1`→SH，`0`→SZ；代码以 `4`/`8` 开头时改为 `.BJ`。

<br>

**输入参数（工具入参）**

名称 | 类型 | 必选 | 描述
---- | ---- | ---- | ----
query | str | Y | 公司名或代码片段，如 `贵州茅台` / `茅台` / `600519`。
limit | int | N | 候选数上限（1–25，默认 10）。

<br>

**返回字段**

每个候选行：

字段 | 描述
---- | ----
symbol | 项目约定 symbol，如 `600519.SH`
name | 名称
source | 固定 `eastmoney`

信封：`{"ok": true, "market": "cn", "source": "eastmoney", "data": {"query","count","candidates":[...]}}`。

<br>

**调用范例**

```python
from tools.stock_market import SearchSymbolTool

print(SearchSymbolTool().execute({"query": "茅台", "limit": 5}, None))
print(SearchSymbolTool().execute({"query": "600519"}, None))
```
