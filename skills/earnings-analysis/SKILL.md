# 盈利预测与分析师修正

两条主线：① **自主预测 vs 一致预期**对比寻找偏差 → SUE/PEAD 策略 ② **跟踪分析师修正动量** → 修正因子是 A 股最稳定的 alpha 之一。

## 一、盈利预测方法

### 自上而下（Top-Down）

```
GDP增速 → 行业增加值增速 → 龙头收入增速 → 利润率 → EPS
```

适用于：行业 beta 判断、大盘盈利周期定位。

### 自下而上（Bottom-Up）

```python
# 量价拆解
revenue = volume * price
# 例：中国神华 = 煤炭销量 × 煤价 + 电力收入

# 客户/产品拆解  
revenue = sum(segment_revenue)
# 例：美的 = 暖通空调 + 消费电器 + 机器人

# 门店/用户拆解
revenue = stores * revenue_per_store
# 例：海底捞 = 门店数 × 翻台率 × 客单价
```

### 利润率假设

- 毛利率：原材料成本占比、产品结构升级
- 费用率：规模效应（收入增→费用率降）、研发投入
- 税率：高新技术 15% vs 普通 25%

## 二、SUE / PEAD 超预期交易

**SUE（标准化未预期盈利）**：
```
SUE = (实际EPS - 预期EPS) / std(预期误差)
SUE > 2σ → 买入并持有 60 天
SUE < -2σ → 卖出/回避
```

**PEAD（盈余公告后漂移）**：利好公告后价格持续上涨 30-90 天，因投资者对信息的反应不足（保守性偏差 + 锚定效应）。

A 股证据：业绩超预期公告后 20 天仍有 3-5% 超额收益。

## 三、分析师修正信号

| 信号 | 强度 | 持续性 |
|------|------|--------|
| 业绩超预期(beat) | ⭐⭐⭐ | 60-90 天(PEAD) |
| 修正广度(上调% - 下调%) | ⭐⭐⭐ | 3-6 个月 |
| 修正幅度 | ⭐⭐ | 1-3 个月 |
| 管理层指引调整 | ⭐⭐⭐⭐ | 立即+30-60天 |
| Whisper number | ⭐ | 1-2 周 |

**修正动量因子**：过去 3 个月分析师上调次数 - 下调次数 → 买入 Top 20%，A 股年化超额 4-6%。

**修正加速**：本月修正幅度 > 前 3 月均值 → 更强信号（修正趋势在加速）。

## 四、A 股业绩预告制度

A 股有**强制业绩预告**制度（净利润 ±50% 必须预告），预告窗口期是重要的超额收益来源。

| 预告类型 | 含义 | 信号 |
|---------|------|------|
| 预增 | 净利润同比增长≥50% | 强看涨 |
| 略增 | 净利润同比增长20-50% | 看涨 |
| 预减 | 净利润同比下降≥50% | 看跌 |
| 首亏 | 首次亏损 | 强看跌 |
| 扭亏 | 扭亏为盈 | 强看涨 |
| 续亏 | 继续亏损 | 看跌 |

## 六、A 股 vs 美股差异

| 维度 | A 股 | 美股 |
|------|------|------|
| 业绩预告 | 强制（±50%） | 自愿 |
| 分析师覆盖率 | 低（小盘几乎无覆盖） | 高 |
| 修正信息传递速度 | 慢（散户主导） | 快 |
| 修正因子的超额 | 4-6% 年化 | 2-4% 年化 |
| PEAD 效应 | 存在但短（20-30天） | 60-90天 |

## 七、常见陷阱

1. **前视偏差**：修正数据要在修正公告日后才能使用
2. **共识样本过小**：小盘股只有 1-2 个分析师覆盖，共识无意义
3. **季节性**：Q4 业绩经常有"洗澡"行为（一次性计提），SUE 失真
4. **分析师偏见**：A 股分析师 95% 是买入评级，看修正幅度不用看绝对评级
